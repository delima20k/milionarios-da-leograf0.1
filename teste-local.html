🔧 PARA TESTE COMPLETO (COM SERVIDOR):

1️⃣ Opção mais fácil - VS Code:
• Instale a extensão "Live Server"
• Clique direito em index.html
• Selecione "Open with Live Server"
2️⃣ Com Node.js:
• Instale Node.js: https://nodejs.org
• Abra terminal nesta pasta
• Execute:
npx serve -s . -p 3000
• Acesse: http://localhost:3000
3️⃣ Com Python:
• Instale Python: https://python.org
• Abra terminal nesta pasta
• Execute:
python -m http.server 8000
• Acesse: http://localhost:8000
📥 Baixar Node.js 🐍 Baixar Python
